CommunityHero AI

## AI-powered Civic Safety Platform

CommunityHero AI is a smart civic safety platform built for the Google Vibe2Ship Hackathon.

### Features
- Report Issues
- Safe Journey
- Building Guardian
- AI-powered Issue Detection
- Real-time Dashboard
- Gemini AI Integration
